Tienda de Productos en PHP
Autor: Ramiro Elortegui Bazán – 5to año
Materia: Programación

Este proyecto es una aplicación simple de una tienda hecha con PHP, HTML, CSS y Bootstrap.
El propósito es cargar productos mediante un formulario, 
guardarlos en un archivo JSON y mostrarlos en pantalla de forma ordenada. 
El trabajo me permitió practicar el uso de formularios, manejo de arrays, 
lectura y escritura de archivos JSON, organización del código, 
estructuras de control y el uso básico de Bootstrap
