Este proyecto es una aplicación web desarrollada en PHP
que permite cargar productos mediante un formulario y almacenarlos 
dentro de un archivo JSON. Luego, esos productos se muestran en pantalla 
utilizando estilos basados en Bootstrap. El proyecto incluye un archivo
 principal llamado index.php y carpetas separadas para los estilos CSS, 
 el código JavaScript y los datos JSON. Para ejecutarlo, 
 la carpeta debe ubicarse dentro de htdocs y abrirse desde el navegador 
 utilizando la ruta localhost. Es necesario que el archivo products.json 
 comience con corchetes vacíos para que la carga funcione correctamente.
